# Netflix Clone 🎬

A pixel-perfect, fully responsive clone of the Netflix India landing page, built with pure HTML and CSS — no frameworks, no JavaScript.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

---

## 📸 Overview

This project replicates the Netflix India landing page UI, including the hero section, feature highlights, FAQ accordion layout, and footer — closely mirroring the real Netflix experience using only semantic HTML and custom CSS.

---

## ✨ Features

- **Responsive Design** — Adapts cleanly across desktop, tablet, and mobile screen sizes
- **Hero Section** — Full-screen background with email sign-up CTA
- **Feature Sections** — Four highlight sections (TV, Mobile Download, Multi-device, Kids Profiles) with images and embedded videos
- **FAQ Section** — Netflix-style expandable FAQ layout
- **Footer** — Complete footer with navigation links, language selector, and India-specific branding
- **Embedded Media** — Autoplay looping videos overlaid on device mockup images
- **Netflix favicon** — Authentic favicon included

## 🚀 Getting Started

No build tools or dependencies required — just open and run.

**1. Clone the repository**
```bash
git clone https://github.com/abhinavshahi27/Netflix-Clone.git
```

**2. Open in your browser**
```bash
cd Netflix-Clone
open index.html
```

Or simply double-click `index.html` to open it in your default browser.
